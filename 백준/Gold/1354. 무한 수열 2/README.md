### 문제 설명

<ul>
	<li>A<sub>i</sub> = 1 (i ≤ 0)</li>
	<li>A<sub>i</sub> = A<sub>⌊i/P⌋-X</sub> + A<sub>⌊i/Q⌋-Y</sub> (i ≥ 1)</li>
</ul>

<p>N, P, Q, X, Y가 주어질 때, A<sub>N</sub>을 구하는 프로그램을 작성하시오.</p>

### 입력

### 출력 

 <p>첫째 줄에 A<sub>N</sub>을 출력한다.</p>
