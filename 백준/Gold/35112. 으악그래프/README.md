### 문제 설명

<p>$N$개의 정점과 $M$개의 간선으로 이루어진 방향성이 없는 단순 연결 그래프 $G$가 주어진다. $G$의 각 정점에는 $1$부터 $N$까지의 번호가 매겨져 있다. 그래프 $G$가 아래의 조건을 만족할 때, $G$는 <strong>으악그래프</strong>다.</p>

<ul>
<li>그래프 $G$에서 임의의 서로 다른 두 간선을 제거했을 때, 그래프가 두 개 이상의 연결요소로 분리된다.</li>
</ul>

<p>두 개의 간선을 제거했음에도 그래프가 연결되어 있는 경우가 하나라도 존재한다면, $G$는 으악그래프가 아니다.</p>

<p>주어진 그래프가 으악그래프인지 판별해보자.</p>

### 입력 

 <p>첫 번째 줄에 정점의 개수 $N$과 간선의 개수 $M$이 주어진다. $\left(3 \le N \le 100\,000;\ N-1 \le M \le 10^6\right)$</p>

<p>두 번째 줄부터 $M$개의 줄에 걸쳐 간선이 연결하는 두 정점의 번호 $u$, $v$가 주어진다. $\left(1 \le u, v \le N;\ u \neq v\right)$</p>

<p>주어지는 그래프는 항상 연결 그래프임이 보장되며, 중복 간선이나 자기 자신을 잇는 간선은 없다.</p>

### 출력 

 <p>주어진 그래프가 으악그래프라면 <span style="color:#e74c3c;"><code>Yes</code></span>, 아니라면 <code><span style="color:#e74c3c;">No</span></code>를 출력한다.</p>
